# dart-cli-api-webscrape-orm-local

## Description
Webscrapes a local nodejs api.
Serialize json data from api to dart object.
Overrides `toString` to print results.

## Tech stack
- dart
  - http

## Docker stack
- google/dart
- nodejs:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
